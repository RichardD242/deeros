# DeerOs

A calm almost low cortisol nature web desktop ui built with react and typescript

## Tech Stack

- **React 19**
- **TypeScript 6**
- **Vite 8**
- **Tailwind Css**

## Dev

```bash
npm install
npm run dev
```

## How to use

- go to [deeros](https://deeros.vercel.app/)
- go through the setup and use the apps
- for more info check out the [manual](https://deeros.vercel.app/manual) subpage

## docs
