# DeerOs

A calm almost low cortisol nature web desktop ui built with react and typescript

## Tech Stack

- **React 19**
- **TypeScript 6**
- **Vite 8**
- **Tailwind Css**

## Dev

```bash
npm install
npm run dev
```
